HTML

  • The head element contains information about the web page.
  • The body element contains content visible to the user.
  • Style, base, link, title and meta are elements found in the head element
  • The style element defines CSS styles used for the website.
  • The base element defines the base URL for the webpage.
  • The link element connects external resources to the webpage
  • The meta element defines things such as the character set, keywords, author and description.
  • The title element defines the title displayed in the webpage tab and is important for search engine optimization.

CSS

  • Inline CSS is code embedded in an HTML file eg the style attribute.
  • An internal style sheet can be embedded in the head element of an HTML file. This will define styles for all defined elements in the HTML file.
  • The margin property defines how much space we want round the outside of an element.
  • The padding property defines how much space to leave around the content inside an element.

Git

  • A feature branch is where we can add new code that does not affect the main branch.
  • Git status checks what branch we are on.
  • Git checkout -b creates a new branch and switches to it.
  • Git add -A adds all code we have created to the next commit. The -A flag tells the add command to add all files.
  • Git commit -m commits the code we have added to the next merge request. The -m flag lets us add a brief description.
  • Git pull origin main downloads the main branch of our code.
  • Git push origin " branch name" uploads code with the defined branch name to Github ready for a pull request and merge.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The if statement executes a block of code if the defined condition is true.
  • An array is a single variable that holds a group of data. The data can be integers, strings, boolean statements or a mixture.
  • An item in an array can be identified with an index number. The first item in an array will be indexed to 0.
  • A for loop uses a single block of code which operates on multiple items in an array.
  • A for loop contains 3 statements; the starting point in the array, the condition which defines whether the for loop will continue running or not and a condition that defines how the loop increments.
